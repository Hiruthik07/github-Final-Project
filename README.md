# Simple Interest Calculator

This is a small Python program to calculate simple interest.

## Formula

Simple interest is calculated as:

```text
SI = (P × R × T) / 100
```

Where:
- `P` is the principal amount
- `R` is the annual interest rate (in percent)
- `T` is the time in years

## Requirements

- Python 3.x

## How to Run

From the `simple-interest-calculator` directory, run:

```bash
python simple_interest.py
```

Then follow the prompts to enter principal, rate, and time.
